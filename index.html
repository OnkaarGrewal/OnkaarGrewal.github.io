01 Software / Control Systems

PID Controller
Simulation Suite

Modelled a second-order dynamic system with a full PID controller in Python, implementing proportional, integral, and derivative tuning to analyse transient response. Real-time Matplotlib visualisation compares system behaviour across varying gain parameters — built as a foundation toward more complex models (quadrotor, spacecraft attitude).

Same PID architecture stabilises satellite orientation, rocket TVC, and re-entry guidance — this is the core loop behind all of it.
