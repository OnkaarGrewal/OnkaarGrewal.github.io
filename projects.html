01 — Software

PID Controller Simulation Suite

Modelled a second-order dynamic system with a full PID controller in Python, implementing proportional, integral, and derivative tuning to analyse transient response. Real-time Matplotlib visualisation compares system behaviour across varying gain parameters — foundation for spacecraft attitude control models.

Same PID loop architecture stabilises satellite orientation, rocket TVC, and re-entry guidance systems.
