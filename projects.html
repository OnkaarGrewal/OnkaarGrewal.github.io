PID Controller Simulation Suite

Developed a PID controller simulation in Python modelling a second-order dynamic system. Implements proportional, integral, and derivative tuning to analyse transient response, with real-time Matplotlib visualisation comparing system behaviour across varying gain parameters. Built as a foundation for progressively complex models — inverted pendulum, drone quadrotor, and eventually spacecraft attitude control.

Same PID loop architecture stabilises satellite orientation, rocket thrust vector control, and re-entry guidance systems.

Portable Weather Station Node

Autonomous weather monitoring system with a sensor suite measuring humidity, temperature, wind speed, and rainfall, with real-time data logging via an Arduino-based embedded system. Sensor communication protocols implemented in C/C++. Mechanical housing designed and fabricated in SolidWorks to protect electronics from environmental exposure while maintaining accurate sensor calibration.

Designing enclosures that maintain sensor accuracy under environmental stress is the same engineering challenge as aerospace instrument bay design.

Wheelchair-Mounted Assistive Launcher

Led mechanical CAD design of a wheelchair-mounted ball launcher achieving 6 m+ range for a client with arthritis. Full SolidWorks assembly developed with structural material selection via Granta EduPack. Coordinated technical design integration across a multidisciplinary team within McMaster's Engineering Design course (ENGINEER 1P13). Design validated for 50 N+ repeated launch loads with accessible, tool-free wheelchair mounting.

Granta EduPack material selection under cyclic load — the same methodology used in aerospace structural component and airframe bracket design.
