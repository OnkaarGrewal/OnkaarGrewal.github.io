01 Software / Control Systems

PID Controller Simulation Suite

Developed a full PID controller simulation in Python modelling a second-order dynamic system. Implements proportional, integral, and derivative tuning with real-time Matplotlib visualisation of system response curves across varying gain parameters. Built as a progressive simulation suite — starting with an inverted pendulum, advancing toward drone quadrotor and spacecraft attitude control models. The simulation uses 4th-order Runge-Kutta integration for accurate ODE solving.

Same PID loop architecture stabilises satellite orientation, rocket TVC, and re-entry guidance systems — this is the foundational control loop behind all of it.

02 Embedded / Hardware

Portable Weather Station Node

Autonomous weather monitoring system with a sensor suite measuring humidity, temperature, wind speed, and rainfall with real-time data logging. Sensor communication protocols implemented in C/C++ on an Arduino-based embedded system using I²C and SPI interfaces. Mechanical housing designed and prototyped in SolidWorks — engineered to protect electronics from environmental exposure while maintaining accurate sensor calibration. The housing design accounts for moisture ingress, thermal expansion, and ventilation flow.

Designing enclosures that maintain sensor accuracy under environmental stress — identical engineering challenge to aerospace instrument bay design.

03 Mechanical / Team Design

Wheelchair-Mounted Assistive Launcher

Led mechanical CAD design of a wheelchair-mounted ball launcher achieving 6 m+ range for a client with arthritis, developed in McMaster's ENGINEER 1P13 capstone. Full SolidWorks assembly with parametric modelling and structural material selection via Granta EduPack — cross-referenced yield strength, density, and corrosion resistance to meet a 50 N+ repeated launch load requirement. Coordinated technical design integration across four sub-teams handling electronics, actuation, user interface, and structure. Design validated through prototype testing with tool-free wheelchair mounting for one-handed operation.

Granta EduPack material selection under cyclic load — the same methodology used in aerospace structural component and airframe bracket design.
