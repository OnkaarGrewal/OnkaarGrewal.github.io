Overview

The ENGINEER 1P13 capstone challenged our team to design an assistive device for a real client — a wheelchair user with arthritis who wanted to play ball sports independently. Our solution was a wheelchair-mounted pneumatic launcher capable of throwing a ball 6+ metres with adjustable direction and force.

I led the mechanical CAD design, modelling the full assembly in SolidWorks and using Granta EduPack to select mounting hardware materials that could handle repeated launch loads without fatiguing or deforming the wheelchair frame.

The project required coordinating across sub-teams handling electronics, user interface, and structural testing — the first time I managed technical design integration in a multidisciplinary setting.

Specifications

Launch Range 6 m+ (adjustable)
Max Launch Load ~50 N
Mounting System Universal wheelchair rail clamp
Frame Material 6061-T6 Aluminium
CAD Software SolidWorks 2024
Material Selection Granta EduPack CES
Team Size 4 engineers

What I Learned

◆
Material selection under load Using Granta EduPack to cross-reference yield strength, density, and corrosion resistance under real-world constraints — not just picking "aluminium" because it's familiar.
◆
Design for the actual user Accessibility constraints forced decisions I wouldn't have made otherwise — tool-free mounting, one-handed operation. Designing for constraints produces better engineering.
◆
Integrating across sub-teams Coordinating CAD with the electronics team's component footprints required constant version control discipline — first time I used SolidWorks in a shared-model context.

Aerospace Connection

Granta EduPack material selection is used throughout aerospace structural design. The same process I used here — defining a performance index, filtering by constraints, and ranking candidates — is how engineers select materials for airframe brackets, satellite bus panels, and launch vehicle skins.

The 50 N launch load analysis also introduced me to how fatigue and cyclic loading are accounted for in design — a core concept in any aerospace structure that sees repeated stress cycles in service.

How to add your 3D model

Export from SolidWorks and upload to your GitHub repo. The viewer supports both .STL and .GLB.

In SolidWorks: File → Save As → select STL (.stl) → Save
In your GitHub repo, create a folder called models/
Upload the file as wheelchair-launcher.stl inside the models folder
Reload this page — the viewer will detect and load it automatically
Optional: for better quality, convert to GLB using free tool at gltf.report and name it wheelchair-launcher.glb
